# Smart-Navigation-Glasses-for-the-Visually-Impaired

##  Overview

A wearable assistive device designed to help visually impaired individuals navigate safely using ultrasonic sensing and real-time feedback.

##  Problem

Visually impaired individuals face challenges in detecting obstacles in their path, especially in unfamiliar environments. Existing solutions can be expensive or inconvenient.

##  Solution

Developed smart glasses equipped with an ultrasonic sensor to detect obstacles and provide real-time feedback using buzzer alerts, enabling safer and more independent navigation.

##  Tech Stack

* Ultrasonic Sensor
* Buzzer Feedback System
* Power/Charging Module

##  Features

* Real-time obstacle detection
* Audio/tactile feedback using buzzer
* Lightweight and wearable design
* Low-cost and energy-efficient

##  How It Works

1. Ultrasonic sensor measures distance to obstacles
2. If an object is detected within a threshold distance
3. Buzzer alerts the user in real-time

##  Impact

* Enhances mobility for visually impaired users
* Provides an affordable assistive solution
* Designed for practical, everyday use

##  Future Improvements

* Add vibration-based feedback
* Integrate with mobile app for GPS navigation
* Use AI for object recognition
